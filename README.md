# Pdf-file-editor-Automation

##📄 PDF Automator: Merge & Extract
A powerful, user-friendly web application built with Streamlit to streamline repetitive PDF tasks. This tool allows users to consolidate multiple documents and pull raw text for analysis in seconds.

##🌟 Key Features
-**Smart Merging**: Combine multiple PDF files into a single, organized document with one click.
-**Instant Text Extraction**: Pull clean text from any PDF for data processing or archival.
-**File Creation**: Instantly automatic file creation with the extracted text.
-**Intuitive UI**: Simple drag-and-drop interface powered by Streamlit—no coding required for the end-user.
-**Fast Processing**: Built using optimized Python libraries for high-speed document handling.

##🛠️ Tech Stack
-**Frontend**: Streamlit (Web Framework)
-**Language**: Python 3.x
-**PDF Libraries**: pypdf for merging and for extraction.
